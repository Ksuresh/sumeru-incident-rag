# Incident INC-2026-1940

## Summary
Increased latency and intermittent 500 errors due to exhausted DB connection pool. Observed after v4.0.0 in US.

## Impact Scope
Requests time out; partial outages for read/write paths. Metric: error rate 18%.

## Observability / Logs
Too many connections for user 'app_user'

## Root Cause (as recorded)
Traffic spike + leak in connection handling after latest release; pool size too small.

## Resolution Steps
Hotfix to close connections; increased pool; scaled DB; restarted affected pods.

## Detection & Resolution Timing
- Time to Detect (TTD): 16m
- Time to Resolve (TTR): 1 Hr

## Preventive Actions
Load test; connection leak detection; set SLO alerts on pool utilization.
